# LogiTrack

Sistema de distribución, inventario y despacho logístico — **Next.js 16 + Supabase**.

## Documentación para el equipo

→ [`docs/HANDOFF-AGENTE.md`](docs/HANDOFF-AGENTE.md)

Incluye: setup, variables de entorno, esquema BD, rutas, auth, RPC de registro, problemas conocidos y pendientes.

Esquema SQL: [`assets/docs/Estructuras de tablas.txt`](assets/docs/Estructuras%20de%20tablas.txt)

## Inicio rápido

```bash
npm install
cp .env.example .env
# Completar NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY, SUPABASE_SERVICE_ROLE_KEY
npm run dev
```
